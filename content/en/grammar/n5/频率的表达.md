1. 体言<sub时间段</sub　次数　动词

   - この花は　二年に　一度　咲きます。

   - 森さんは　二週間に　一回　家へ　帰ります。

2. 「回」和「度」的区别：

   | 回                 | 度                                   |
   | ------------------ | ------------------------------------ |
   | 第/全~回           | 一般不用（有特例）                   |
   | 单纯的数字         | 深寒特别意义，如第30度庆祝结婚纪念日 |
   | 给人感觉有下一次   | 给人感觉不一定能有下一次             |
   | 可以用来表示大数字 | 一般不用来表示大数字                 |

3. 当时间体言单位数量为1时，可以省略数字

   - 一年に五回　→　年に五回
   - **一か[月]^(げつ)に**三回　→　**[月]^(つき)に**三回
   - 一週間に二回　→　週に二回
