|              | 简体               | 敬体                       |
| ------------ | ------------------ | -------------------------- |
| 非过去时肯定 | きれいだ           | きれいです                 |
| 非过去时否定 | きれいではない     | きれいではありません       |
| 过去时肯定   | きれいだった       | きれいでした               |
| 过去时否定   | きれいではなかった | きれいではありませんでした |

- これは本です。　これは本だ。
- 今日は暇です。　今日は暇だ。

 注意区分上面四个例句，很相像，但，是不同语法。
