- カ変動詞：来る　→　くれ
- サ変動詞：する　→　すれ
- 一段動詞：去「る」＋「れ」
  - 見る　→　見れ
- 五段動詞：「う」段　→　「え」段
  - 買う　→　買え
  - 待つ　→　待て
  - 売る　→　売れ
  - 読む　→　読め
  - 遊ぶ　→　遊べ
  - 死ぬ　→　死ね
  - 書く　→　書け
  - 泳ぐ　→　[泳]^(およ)げ
  - 話す　→　話せ
