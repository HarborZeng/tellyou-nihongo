译为：在趁...之际、借...时机

- お[降]^(お)りの**際**には、お忘れ物がないようにご注意ください。
- お会いして**際**に、お渡しいたします。
- 転職する**際**の必要な手続きの準備ができた。
- 先月帰した**際**に、高校の先生を[訪]^(おとず)れた。

|      | 時     | 際           | 場合               |
| ---- | ------ | ------------ | ------------------ |
| 接续 | 连体形 | 动简、名の   | 连体形             |
| 时间 | 无限制 | 不用于长时间 | 不用于长、过去时间 |
| 含义 | 时候   | 时候         | 时候、场合         |

- 地震の**時 ／際 ／場合**は、エレベーターを使わないです。
- 子供の**時**よく[祖母]^(そぼ)と一緒に公園へ行きました。
- 先月帰した**時／際**に、高校の先生を[訪]^(おとず)れた。
- 子供の**場合**、入場料は要りません。
